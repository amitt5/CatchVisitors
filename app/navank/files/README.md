# NAVANK Image URLs

All images scraped from https://www.navank.in
These are hosted on their WordPress — download and store locally before building.

## Instructions
To download all images, run in terminal:
```
while IFS= read -r url; do wget "$url" -P ./images/; done < image_urls.txt
```
